how we use the archive

dear user,

welcome to team 3128’s past game database. thank you for stopping by! our strategy department uses this archive to explore precedents for elements in new games by referencing old ones. here, you will find a compiled page of robot archetypes for every game dating back to 2010. you’ll also find strategical commentary, which consists of our thoughts on each game and how we approached analyzing it. we hope that this archive helps anyone perusing it and if you have any questions, please don’t hesitate to contact us!

thank you,
